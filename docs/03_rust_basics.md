# Rust Basics (Beginner)

## Core concepts
- Ownership: each value has one owner; avoids data races.
- Borrowing: references let you use data without owning it.
- Structs: custom data types for grouping fields.
- Enums: define a type with multiple variants.
- Vec: dynamic arrays.
- Result and Option: safe error handling and nullable types.
- Cargo: Rust build tool and package manager.
- Modules: organize code in files and folders.

## Why Rust here
We need fast backtests on large datasets. Rust is fast, safe, and integrates with Python via PyO3.
